# Gamers SNS

## サイト概要
様々なゲームのスクリーンショットを、一言と共に共有することができる

ゲーム好きに特化したSNS。

### サイトテーマ
すごい事や、面白い出来事の共有。

様々なゲームの攻略方法の共有。

### テーマを選んだ理由
自分自身がゲームが好きなこと。

自分では見つけられなかった、攻略法の共有。

eスポーツが盛んになりつつある現代で、

ゲームに特化したSNSがあっても良いと思いました。

### ターゲットユーザ
ゲームが好きな人。


### 主な利用シーン
Twitter、Instagram等、SNSと同じ。
